# More Modelling Recipes

A collection of data science and actuarial modelling recipes. Simpler posts include reproducible code examples, more complex posts discuss architecture and design.

Live site: https://pat-reen.github.io/more-modelling-recipes/

## Topics covered

- Regression and GLMs
- Bayesian statistics
- Decision trees, random forests, XGBoost
- Natural language processing and topic modelling
- Price elasticity and demand modelling
- Data transformations and simulation
- Network diagrams and visualisation
- Life insurance industry statistics

## Tech stack

- [R](https://www.r-project.org/) / [R Markdown](https://rmarkdown.rstudio.com/)
- [Distill](https://rstudio.github.io/distill/) for the website framework
- Hosted on [GitHub Pages](https://pages.github.com/)
